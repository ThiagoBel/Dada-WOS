Documentação dos Comandos do Dada

O sistema Dada suporta diversos comandos que podem ser digitados no terminal para executar ações específicas.

Terminal

1. Comandos de Sistema

// reset$$dada$$%
Descrição: Reinicia o sistema, apagando algumas configurações.
// reset#cmd#in#enter$$dada$$%
Descrição: Reinicia o cmd toda vez que executa.
// no#reset#cmd#in#enter$$dada$$%
Descrição: Não reinicia o cmd toda vez que executa.
// reload#but#no#load$$dada$$%
Descrição: Recarrega a página pulando a tela de carregamento.

2. Comandos de Aparência

// remove#css$$dada$$%
Descrição: Remove os estilos CSS da página.
// remove#cmd$$dada$$%
Descrição: Remove o terminal da página.
// rmvid <ID>
Descrição: Remove algo com o ID relacionado.
// rmvcs <CLASS>
Descrição: Remove algo com a CLASS relacionado.
// rmvqr <ALGO>
Descrição: Remove qualquer coisa que você digitou, exemplo 'body'.

3. Comandos de Configuração

// tester#mode$$dada$$%
Descrição: Ativa o modo de teste, destacando os elementos da página com bordas vermelhas.
// exit#tester#mode$$dada$$%
Descrição: Sai do modo de teste.

4. Comandos de Importação e Scripts

// import <arquivo.js>
Descrição: Importa e executa um arquivo JavaScript externo/interno.
// import#perm <arquivo.js>
Descrição: Importa permanentemente e executa um arquivo JavaScript externo/interno.
// reset#imports#perm$$dada$$%
Descrição: Remove todos os imports.
// noimport#perm <arquivo.js>
Descrição: Remove o import específico.

5. Comandos de Informações

// la$$dada$$%
Descrição: Lista todas as divs do sistema.
// li$$dada$$%
Descrição: Lista todas as importações do sistema.
// lf$$dada$$%
Descrição: Lista qual favicon está sendo usado (usado pelo comando do sistema).
// ll$$dada$$%
Descrição: Lista todos os localStorages do sistema.

6. Favicon

// favicon <arquivo.ico>
Descrição: Muda o favicon.
// favicon#perm <arquivo.ico>
Descrição: Muda o favicon permanentemente.
// nofavicon$$dada$$%
Descrição: Remove o favicon.

7. Comandos Diversos

// title <novo título>
Descrição: Altera o título da página.
// title#perm <novo título>
Descrição: Altera o título da página permanentemente.
// notitle$$dada$$%
Descrição: Remove o titulo da página (deixa como Dada ou outro se tiver com outro sistema embutido).
// random [min] [max]
Descrição: Gera um número aleatório.
// calc <expressão matemática>
Descrição: Calcula uma expressão matemática, exemplo: 1+1.
// time$$dada$$%
Descrição: Mostra o horário.
// clear#cmd$$dada$$%
Descrição: Limpa o terminal.
// clear#output$$dada$$%
Descrição: Limpa o output (põe 0).

8. Terminal output

// dt <mensagem>
Descrição: Valor do terminal.
