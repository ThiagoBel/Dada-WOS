Dada


Olá!

Isso é uma guia de ajuda para pessoas que não sabem usar o Dada ou desconhece o Dada.


O que é o Dada:

Dada é um web sistema operacional, que é 100% open-source e 100% personalizável.


Como usar o terminal:

Você precisa usar o Javascript para usar o terminal.


Como abaixar bibliotecas:

Use o comando: // import NOME.js ou // import#perm NOME.js.


Como usar as bibliotecas do Dada:

esse é o caminho: Mods: models/mods/NOME.js, Texturas: models/texture/NOME.js.
